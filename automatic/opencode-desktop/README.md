# OpenCode Desktop

OpenCode Desktop is the beta desktop application for OpenCode, an open-source AI coding agent designed to assist developers in writing code efficiently across various environments, including terminals, IDEs, and now desktop applications.

## Features

- **LSP Enabled**: Automatically loads and integrates the right Language Server Protocols (LSPs) for optimal AI assistance.
- **Multi-Session Support**: Start multiple AI agents in parallel on the same project for concurrent tasks.
- **Share Links**: Generate shareable links for sessions, references, or debugging purposes.
- **Claude Pro Integration**: Log in with Anthropic to access Claude Pro or Max accounts.
- **Any Model Support**: Connect to 75+ LLM providers through Models.dev, including Claude, GPT, Gemini, and local models.
- **Any Editor Compatibility**: Available as a terminal interface, desktop app, and IDE extensions for VS Code, Cursor, Zed, Windsurf, and VSCodium.
- **Privacy First**: Operates in privacy-sensitive environments without storing code or context data.
- **Open Source**: Fully open-source with over 50,000 GitHub stars and contributions from 500+ developers.

## Usage

After installation, launch OpenCode Desktop from your applications menu. Connect your preferred AI models and start coding with AI assistance.

For more details, visit the [official documentation](https://opencode.ai/docs).
